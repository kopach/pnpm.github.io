pnpx CLI

新規ユーザー向け​

pnpx (PNPm eXecute) は、依存関係としてインストールせずにレジストリからパッケージを取得し、それを一回限りで読み込んで、 そのパッケージのデフォルトのコマンドバイナリーを実行するコマンドラインツールです。

例えば、 create-react-app を別のプロジェクトでインストールせずに、どこでも React アプリをブートストラップすることができます。

pnpx create-react-app my-project

レジストリから create-react-app が取得され、指定された引数を使用して実行されます。 詳細については、同じインターフェイスを提供する npm の npx を参照すると良いでしょう。こちらは内部で pnpm ではなく npm を使用している点が異なります。

npm ユーザー向け​

npm has a great package runner called npx. pnpm offers the same tool via the pnpx command. The only difference is that pnpx uses pnpm for installing packages.
